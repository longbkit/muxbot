# 发布说明

## 目的

`docs/releases/` 是 `clisbot` 标准的版本历史。

当前中文只完成了这一页入口。具体 release note 目前仍先链接到英文原文。

它主要回答：

- 这个版本改了什么
- 对用户和运维者的影响是什么
- 以后更新与验证时哪些事实最重要
