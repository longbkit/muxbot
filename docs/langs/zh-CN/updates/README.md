# 更新说明

## 目的

`docs/updates/` 是面向用户和运维者的更新文档层，重点回答：

- 现在最该关注什么
- 升级后应该马上开始用什么
- 正确的升级路径是什么
- 当前运维风险有哪些变化

当前中文只完成了这一页入口。更深的更新说明暂时仍然回退到英文原文。

## 主要文件

- [update-guide.md（英文原文）](../../../updates/update-guide.md)
- [v0.1.45 release guide（英文原文）](../../../updates/releases/v0.1.45-release-guide.md)

## 什么时候该看这里

- 升级 `clisbot` 之前
- 想快速知道“最近值得关注的变化”
- 想判断某个 release 是否改变了 setup 或 ops 方式

## 基本规则

- 版本事实以 `docs/releases/` 为准
- manual action 判断从 `docs/migrations/index.md` 开始
- `updates` 文档应该短、明确、可执行
