# Hướng dẫn update

## Mục đích

`docs/updates/` là nhóm tài liệu hướng tới người dùng để trả lời:

- lúc này cần quan tâm gì
- nên bắt đầu dùng gì
- update path nào đúng
- rủi ro vận hành nào vừa thay đổi

## File chính

- [update-guide.md](./update-guide.md): file `clisbot update` dùng
- [v0.1.50 release guide](./releases/v0.1.50-release-guide.md): bản tóm tắt ngắn cho release lớn

## Khi nào đọc nhóm này

- trước khi update `clisbot`
- khi muốn brief nhanh “có gì mới”
- khi muốn biết release nào đủ lớn để đổi setup/ops flow

## Nguyên tắc

- thông tin chuẩn về phiên bản nằm ở [ghi chú phát hành](../releases/README.md)
- quyết định có cần thao tác tay hay không nằm ở `docs/migrations/index.md`
- nhóm `updates` phải ngắn, opinionated, và dễ hành động
